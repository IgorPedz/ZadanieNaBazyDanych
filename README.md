## Przed uruchomieniem
 - Folder scripts - wypakuj całą zawartość w folderze xampp/htdocs.
 - Stworzenie Bazy mySqL za pomocą struktury zawartej na githubie.
 - Włączenie localhosta xampp
 2. Uruchomienie aplikacji
 ```bash

   npm install

   npm start
```
